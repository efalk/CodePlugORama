Anytone 878 schema

May work with other Anytone radios

In the sample files I found, every single field was quoted. This doesn't
seem to be a requirement.

Note: the software that uses this pops up a warning about how this format is
subject to change. I've already had to change the format
once. This is likely not worth pursuing any more.
